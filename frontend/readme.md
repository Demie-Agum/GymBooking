to run frontend 
php -S localhost:3000

to run backend
cd backend
php artisan serve
# Backend will run on http://127.0.0.1:8000

IMPORTANT: 
- Frontend runs on port 3000 (http://localhost:3000)
- Backend runs on port 8000 (http://127.0.0.1:8000)
- Make sure both servers are running before testing login
